¿Qué es HTML y cuál es su función principal en el desarrollo web?¿Qué es HTML y cuál es su función principal en el desarrollo web?

HTML es un lenguaje de hipertexto cuya función en el desarrollo web es estructurar la página web a través de etiquetas que cumplen con diferentes funciones que brindan determinados elementos esenciales para el diseño web.

Last updated 3 mins ago

¿Qué son las etiquetas HTML y cómo se utilizan?

Las etiquetas de HTML son una especie de instrucciones que realizando funciones de diseño y estructura específicas que actuan como comandos que realizan acciones al cargarse la página web.

Last updated 3 mins ago

Proporciona ejemplos de algunas de las etiquetas HTML más comunes.

Algunas de las etiquetas más comúnes son (entre llaves): p, h, body, header, footer, div, img, br, link, nav, hr, ol, ul, il, a.

Last updated 3 mins ago

¿Qué es CSS y cómo se utiliza para dar estilo a una página web?

CSS es un lenguaje que se utiliza para dar diseño a una página web de una manera más específica y perso nalizada, su funcionamiento se basa en seleccionar las etiquetas que se van a editar a través de su clase o ID y dentro de llaves colocar las intrucciones que queremos darle a las determindas etiquetas.

Last updated 3 mins ago

¿Cuáles son las propiedades más importantes de CSS?

Las propiedades más importantes de CSS son el modelado de cajas (Box model), flexbox, el diseño responsivo estilo y colores, tipografías, posicionamiento y diseño y animaciones o transiciones.

Last updated 3 mins ago

Explica el modelo de caja (box model) en CSS

El modelado de caja es una herramienta que ayuda a posicionar los elementos en una página web, se conforma por las propiedades de margin, border y padding donde a través de estos se posicionan varios elementos.

Last updated 3 mins ago

¿Qué es el diseño responsivo y cómo se logra con CSS?

El diseño responsivo es una herramienta de CSS que ayuda a que los elementos de la página cambien su tamaño de acuerdo al las proporciones dle dispositivo que lo ejecute, se logra a través de @media donde se estipula el tamaño del dispositivo al que se le aplicarán las propiedades que se marquen.

Last updated 3 mins ago

¿Qué es JavaScript y cómo se integra en el desarrollo web?

Javascript es un lenguaje de programación web que se adapta muy fácilmente a HTML y otras herramientas de desarrollo web, en el desarrollo web se integra a través de su propia etiqueta de script en la que através del DOM se pueden programar distintas acciones.

Last updated 3 mins ago

Explica cómo JavaScript interactúa con el HTML y CSS a través del DOM (Document Object Model).

EL DOM se carga automáticamente cuando se abre la página web e interactúa con HTML y CSS a través de las intrucciones para tomar elementos y etiquetas ya sea por su ID (getElementById) o por su clase (getElementsByClassName).

Last updated 3 mins ago

Proporciona ejemplos de cómo JavaScript puede hacer que una página web sea interactiva.

JavaScript puede hacer interactica a una página a través de botones que realicen determinadas acciones al seleccionarlos, creando animaciones de distintos elementos, mostrar o ocultar elementos, cambiar propiedades, acordeones, calculos de operaciones, etc.

Last updated 3 mins ago

¿Qué es Git y por qué es importante para el desarrollo web?

Git es una herramienta que ayuda a tener un mayor control de las versiones de nuestros proyectos y facilita el trabajo en equipo cuando el repositorio se aloja en línea a través de GitHub.

Last updated 3 mins ago

Explica cómo Git se usa para el control de versiones en proyectos de desarrollo web.

Git se utiliza para el control de versiones a través de la herramienta de Git Bash que a través de distintos comandos se pueden ir registrando distintas versiones del proyecto ayudando a que esté más organizada.

Last updated 3 mins ago

¿Qué es GitHub y cómo facilita la colaboración en proyectos web?

GitHub es una herramienta que sirve como servidor del proyecto para ir almacenando las distintas versiones del proyecto y posteriormente subiendola a la plataforma, en la cual se puede trabajar en equipo a través de ramas de trabajo facilitando la colaboración de proyectos web.

Last updated 3 mins ago
